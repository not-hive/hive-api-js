# Hive Home

A wrapper for the Hive Home API.

Version 1.1

## Installation
Node.js
```bash
$ npm i hive-home
```
In the browser
```html
<script src="https://cdn.jsdelivr.net/npm/hive-home@1/dist/hive-home.min.js"></script>
<script>
  console.log('This is HiveHome version', HiveHome.VERSION);
</script>
```

## Development
Test
```bash
$ npm run test
```
Build
```bash
$ npm run build
```
